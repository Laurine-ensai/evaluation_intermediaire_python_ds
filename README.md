# evaluation_intermediaire_python_ds
Exploitation de données électorales avec Python. Evaluation intermédiaire Python pour la data science (mi-semestre 2026).

# Analyse des données électorales

## Installation

Installer les dépendances :

```bash
pip install -r requirements.txt
```

---

## Utilisation

Ouvrir le notebook 

Puis cliquer sur **Run all**.

---

## Reproductibilité

Le projet est entièrement reproductible : toutes les dépendances sont listées dans `requirements.txt` et les données sont importées automatiquement.
